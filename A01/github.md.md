# Configuració i ús de GitHub  
  
## Introducció  
  
En aquesta activitat he après a crear i configurar un compte de GitHub, així com utilitzar GitHub Desktop per gestionar repositoris i pujar arxius al núvol.  
  
GitHub és una plataforma que serveix per guardar projectes de programació online i controlar els canvis del codi.  
  
---
## Creació del compte de GitHub  
  
Primer vaig entrar a la pàgina oficial de GitHub:  
  
https://github.com  
  
Després vaig crear un compte introduint:  
- Nom d’usuari  
- Correu electrònic  
- Contrasenya  
  
Quan el compte ja estava creat, vaig verificar el correu electrònic.  
  
---  
  
## Instal·lació de GitHub Desktop  
  
Després vaig instal·lar GitHub Desktop des de:  
  
https://desktop.github.com/  
  
Aquest programa facilita molt treballar amb GitHub perquè permet fer commits i pujades de fitxers amb una interfície gràfica.  
  
---  
  
## Crear un repositori  
  
Un cop dins GitHub vaig crear un nou repositori.  
  
Vaig seleccionar:  
- Nom del repositori  
- Repositori públic  
- Crear README  
  
Després vaig clonar el repositori al meu ordinador utilitzant GitHub Desktop.  
  
---  
  
## Ús de Markdown  
  
Per redactar les entrades he utilitzat fitxers `.md` (Markdown).  
  
Markdown és un llenguatge senzill que permet:  
- Crear títols  
- Afegir imatges  
- Escriure codi  
- Fer llistes  
  
Per editar aquests arxius he utilitzat Obsidian.  
  
---  
  
## Funcionament bàsic de GitHub  
  
Els passos que he seguit han estat:  
1. Modificar els arxius del projecte  
2. Guardar els canvis  
3. Fer un commit  
4. Fer push al repositori online  
  
Això permet tenir còpies de seguretat i controlar les versions del projecte.
